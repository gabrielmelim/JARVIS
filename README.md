### Módulo IntroducaoJava

#### Descrição

Neste módulo, que faz parte do branch **IntroducaoJava**, trabalhamos com o conceito básico de Java e a criação de um programa simples. O objetivo principal aqui é introduzir o ambiente de desenvolvimento e os fundamentos da linguagem Java.

#### O que foi feito

- **Criado um arquivo `Main.java`**: Este é o ponto de entrada do programa Java.
- **Código**: O código abaixo é o exemplo clássico de um programa "Olá Mundo!" em Java, que serve para demonstrar a estrutura básica de um programa Java e o processo de execução.

```java
public class Main {

    public static void main(String[] args) {
        System.out.println("Olá mundo!");
    }
}
```

#### Explicação do Código

- **`public class Main`**: Define uma classe pública chamada `Main`. Em Java, todos os códigos devem estar dentro de uma classe.
- **`public static void main(String[] args)`**: Este é o método principal (`main`) que serve como ponto de entrada para a execução do programa. Todo programa Java começa a execução a partir deste método.
- **`System.out.println("Olá mundo!");`**: Este comando imprime a mensagem "Olá mundo!" na tela. É usado para verificar se o ambiente está configurado corretamente e para se familiarizar com a sintaxe básica do Java.
